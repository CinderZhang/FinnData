Python for Financial Data Analysis

This learning path is designed to teach you how to use Python for financial data analysis. You will learn the fundamentals of Python programming, how to work with financial data, and how to perform various financial analyses using Python.

Each week, you will learn a new topic and apply your knowledge to a real-world financial analysis project.

Learning Path (13 Weeks)

  • Week 0: Warming Up
  • Week 1: Introduction to Python and Setting Up the Environment
  • Week 2: Python Basics: Variables, Data Types, and Operations
  • Week 3: Control Structures: Loops and Conditional Statements
  • Week 4: Functions and Modules in Python
  • Week 5: Python Data Structures: Lists, Tuples, Sets, and Dictionaries
  • Week 6: Working with Financial Data: Introduction to Pandas
  • Week 7: Data Cleaning and Preprocessing
  • Week 8: Data Visualization with Matplotlib and Seaborn
  • Week 9: Time Series Analysis
  • Week 10: Portfolio Optimization
  • Week 11: Algorithmic Trading Strategies
  • Week 12: LLM and Prompt Engineering
  • Week 13: Final Project: Building a Financial Analysis Application
